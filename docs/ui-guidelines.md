# UI Guidelines COCOS

## Брендові токени

### Кольори
- `--bg-cream: #efece0` - основний фон.
- `--bg-cream-soft: #e7e2d3` - підкладки блоків.
- `--bg-circle: #d8d0b4` - декоративні кола у фоні.
- `--brand-green-dark: #324426` - основний фірмовий зелений.
- `--brand-green-deep: #24361f` - темний зелений градієнтний акцент.
- `--brand-green-mid: #6f8850` - фокус-стани та підсвітка.
- `--text-dark: #2c201a` - основний текст.
- `--text-muted: #5a4b42` - другорядний текст.
- `--danger: #8f3434` - помилки.
- `--success: #3f6232` - успішні стани.

### Типографіка
- `--font-heading: 'Montserrat', sans-serif` - заголовки, кнопки, акценти.
- `--font-body: 'Nunito', sans-serif` - основний текст.

### Радіуси та тіні
- `--radius-sm: 12px`
- `--radius-md: 18px`
- `--radius-lg: 26px`
- `--radius-pill: 999px`
- `--shadow-soft` - кнопки/липкі елементи.
- `--shadow-card` - картки/порожні стани.

## Responsive-брейкпоінти

- Mobile: `max-width: 480px`
- Small tablet: `481px - 768px`
- Tablet/Desktop: `min-width: 769px`

Основна контейнерна ширина задається змінною `--page-max-width`.

## Правила компонентів

### Header
- Липкий (`position: sticky`) з фоном і легким blur.
- Логотип береться тільки з `public/logo.png`.
- Кнопка кошика кругла, з badge для кількості позицій.

### Main nav (`Кухня/Бар`)
- Завжди 2 кнопки з активним станом.
- Активна кнопка має зелений градієнт і світлий текст.

### Social links
- Блок `social-links` між main-nav і category-nav.
- Кнопки з SVG-іконками, відкриття в новій вкладці (`target="_blank"` + `rel="noopener noreferrer"`).

### Category nav
- Тільки багаторядковий режим (`wrap`), без горизонтального скролу.
- Активна категорія контрастна (темний фон, світлий текст).
- На клік змінюється тільки активний стан кнопок, без повного re-render при кожному виборі категорії.

### Список позицій меню
- Назва + лідер-точки + ціна.
- На мобільному лідер може ховатися, щоб не ламати читабельність.
- Після додавання позиції в рядку показується inline-контрол `- / кількість / +`.

### Модальне вікно кошика
- Відкриття з блокуванням скролу body (`body.modal-open`).
- `max-height` на `dvh`, футер враховує safe-area iOS.
- Кнопки +/- для кількості через делегування подій.

### Стани
- Loader: обов'язково анімований `loader-spinner`.
- Empty/cart state: стилізований контейнер із поясненням.
- Error state: окремий стиль (`menu-error`), без inline стилів.
- Toast status: `status-toast` для non-blocking повідомлень.

## Accessibility

- Не вимикати масштабування сторінки у viewport.
- Використовувати `:focus-visible` для всіх інтерактивних елементів.
- Модальне вікно закривається по `Escape` і по кліку на overlay.
- Для користувачів із `prefers-reduced-motion: reduce` відключати важкі анімації/transition.
