# Viber Ordering (Static, GitHub Pages)

## Підхід

Сайт працює повністю без бекенду:
- формує текст замовлення локально у браузері;
- копіює текст у буфер (best-effort);
- відкриває Viber чат на номер кафе.

Немає:
- Viber Bot API,
- webhook,
- токенів,
- серверних запитів.

## Номер та deep link

- Viber номер кафе: `+380972991794`
- Deep link: `viber://chat?number=%2B380972991794`

## UX потоку

1. Клієнт заповнює:
   - ім'я (обов'язково),
   - телефон (обов'язково),
   - додає позиції (мінімум 1).
2. Бачить `Message preview` перед надсиланням.
3. Натискає `Order in Viber`.
4. Сайт:
   - копіює текст,
   - відкриває Viber чат.
5. Якщо копіювання не вдалося:
   - показується модальне fallback-вікно,
   - кнопки `Copy message` і `Open Viber`.

## Технічні функції

- `buildOrderMessage(orderState)`
- `copyToClipboard(text)`
- `openViberChat()`

## Clipboard fallback

1. `navigator.clipboard.writeText`.
2. Якщо помилка: виділення `textarea` preview + `document.execCommand('copy')`.
3. Якщо знову помилка: залишається виділений preview + коротка інструкція ручного копіювання.

## Якщо Viber не встановлений

У футері модального вікна завжди показано:
- номер `+380972991794`,
- кнопку `Copy phone number`.

Користувач може вручну відкрити Viber і вставити текст у чат.

## Перевірка

1. Додайте позиції в кошик.
2. Заповніть ім'я і телефон.
3. Натисніть `Copy message` — текст має копіюватися.
4. Натисніть `Order in Viber` — чат Viber має відкриватися.
5. На обмежених браузерах перевірте fallback modal.
